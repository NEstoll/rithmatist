A game based on the novel "The Rithmatist".
Current Features:
    Custom collision detection
    Custom networking/multiplayer code
    3 lines currently implemented, with custom behaviors and rendering
    Custom mouse control, limiting mouse movement based on lines drawn

Planned Features:
    2 additional lines, with additional features being added to existing lines
    Menuing/UI work
    Bind points/more line interactions
    Matchmaking/better multiplayer integration
    Add better sprites/textures
